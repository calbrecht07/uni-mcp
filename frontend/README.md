# Frontend (UNI MCP)

React dashboard for UNI MCP. It handles authentication, onboarding, provider connection flows, and chat/profile interfaces that call the backend API.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- Environment values must stay in local `.env` files and are not committed.
- Shared app architecture and repo-wide setup live in the root `README.md`.
